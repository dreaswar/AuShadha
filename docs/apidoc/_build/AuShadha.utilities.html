AuShadha.utilities.queries.can_add_new_visit(patient)[source]¶

Queries whether a given patient Can add a new visit. Takes a patient instance as a argument. Basically checks whether he/she has an active admission.

If the visit adding model is changed and new visit addition is not
permitted if an active visit is there that logic can also go here

Returns Boolean
